## Developer Getting Started

download this project , run dev after install deps.

follow tauri quickstart guide before run this project .

```
$ git clone https://github.com/lifecoder1988/tauri-watch-coin.git
$ cd ./tauri-watch-coin
$ yarn install
$ yarn tauri dev

```
